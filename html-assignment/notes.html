5 HTML Tags I have learnt and their descriptions.

Headings (h1) tag
This is described as the most important heading of the page.
Emphasis (em) tag
This is described as the tag that makes a text a bit more important.
Doctype (!doctype HTML) tag
This is the first tag that needs to be added to our document. It states the nature of the document we are working on.
Anchor (a href="") tag
This is described as the tag that helps create links on our webpage.
Table header (thead) tag
This is described as the tag that helps to add headings in a row format for the table columns.
